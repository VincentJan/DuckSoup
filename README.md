# DuckSoup

## 介绍

DuckSoup是基于Ncurses制作的命令行孤岛生存游戏。玩家操控角色在资源与危险并存的孤岛上生存，建造住所，发展农牧业，甚至工业。

## 游戏玩法

2d俯视视角下控制角色移动，战斗和建造。没有胜利条件，玩家只需要活下去，直到死亡（意外，或是寿命耗尽）。
